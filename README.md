# pong in html/css/js
Simple pong game

## KeyBinds
| Key | Action |
| Enter | Start Game |
| Paddle 1 | Keys |
| W | Up |
| S | Down |
| Paddle 2 | Keys |
| Arrow Up | Up |
| Arrow Down | Down |

In Script.js
u can change the speed of the paddle movement and the speed of the ball
